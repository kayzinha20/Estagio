# Conteúdo de Estágio – Python

Repositório com exercícios práticos em Python, desenvolvidos para treinar lógica de programação e fundamentos da linguagem.

## Conteúdos abordados
- Entrada e saída de dados
- Estruturas condicionais
- Funções em Python
- Operações matemáticas
- Listas e laços de repetição
- Validação básica de dados

## Exercícios

- Divisão com validação
- Validação simples de CPF
- Cálculo de IMC
- Soma de elementos de uma lista
